h2. dot-files

This is a modest collection of my dotfiles and plugins for

    * bash
    * git
    * tmux
    * signatures
    * ssh  
    * sup  
    * vim


h3. Requirements

    * Vim 7.3+
    * Vim compiled with Python (2.4+) support


h3. Installation

Clone the repository and link files into home. To include bash configurations use:

bc.. for m in alias prompt sprunge keychain; do
    . "~/shared-dotfiles/bash/$m"
done
