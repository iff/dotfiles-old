h2. dot-files

This is a modest collection of my dotfiles for

    * bash
    * git
    * mutt 
    * newsbeuter  
    * screen
    * signatures
    * ssh  
    * sup  
    * vim

h3. Installation

Clone the repository and link the files into home.. For bash configurations use:

    for m in alias prompt sprunge keychain; do
        . "~/shared-dotfiles/bash/$m"
    done
